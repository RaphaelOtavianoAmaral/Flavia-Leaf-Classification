# Flavia-Leaf-Classification
Repository with Python scripts to classify leaves from Flavia Leaf Dataset

# Dataset Oficial Download
Site: http://flavia.sourceforge.net/
Direct download(1GB): https://sourceforge.net/projects/flavia/files/Leaf%20Image%20Dataset/1.0/Leaves.tar.bz2/download
